A tag ol serve para criar uma lista ordenada de itens. Ou seja, os itens estarão enumerados conforme forem postos.

Tag ul

Voltar ao Início

A tag ol serve para criar uma lista não ordenada de itens. Ou seja, serão dispostos sem ordem enumerada e apenas com 'pontos' organizando-os.
